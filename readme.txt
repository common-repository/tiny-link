Download `Tiny Link` plugin and upload it to `/wp-content/plugins/` directory2. Activate the plugin through the `Plugins` menu in WordPress3. Place `<?php TinyLink() ?>` in your templates== Frequently Asked Questions ==NA== Screenshots ==NA
